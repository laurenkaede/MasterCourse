Unit 1, Outcome 2

2.1 - Review the Ongoing Use of IT Tools and Change Approach as Needed

I was very happy with the process of using Trello throughout the creation of my digital project. It was easy to use, navigate and personalise to meet the needs of my website project. The Kanban method is an extremely useful planning tool that Trello incorporates well, and it helped to keep me on target and organised. I found enabling a due date for a given task to be very useful as it ensured I kept to my timeline.

Visual Studio Code is an excellent piece of software that allowed me to create my digital project. It’s clearly organised and easy to use for beginners. If there’s any errors in your code, it flags up and offers a suggestion to fix the issue which I found really useful.

Github is a website that enables the user to upload their digital projects so that they’re accessible anywhere. I found this especially useful as it meant that I wasn’t restricted to working from one laptop, and one location. The version control aspect was excellent and helped me to stay organised with my digital project.
