Unit 1, Outcome 2

2.2 - Describe Whether the IT Tools Selected Were Appropriate

Trello utilises the use of the Kanban method, which I found was a very appropriate way to plan my digital project. It enabled me to plan out the tasks I needed to take in a step by step process, and allocate them to separate lists. This allowed me to stay very organised throughout my digital project as I could easily see which tasks still needed to be started, which tasks were in process, and which tasks were complete.

Visual Studio Code was an appropriate IT tool to use in the creation of my digital project. It enabled me to use HTML code to create the structure of my website, and CSS to style my website.

Github allows the user to upload their digital projects, and view them on a live website. This was an appropriate way to manage version control, and have any changes updated. These changes were easily organised with commit messages.
