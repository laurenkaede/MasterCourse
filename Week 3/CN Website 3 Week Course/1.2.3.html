Unit 1, Outcome 2

2.3 - Assess the Strengths and Weaknesses of Your Final Project

Overall I’m very happy with how my website turned out. The strengths of my website include the navigation bar which enables the user to access all parts of my website from any location on my website. Another strength would be the images used with the incorporation of the transparency overlay. This allows the text on my website to be easily read which makes it more accessible. I also found that making a separate page per question made my website more organised which also increases accessibility.

One of the weaknesses of my website is the text margins. I feel that the text is too spread from one side of the page to another, which can decrease readability. This is something that I would like to improve but unfortunately wasn’t able to do due to time constraints. Another weakness of my website was how I coded the navigation bar. The code used ended up conflicting with other features that I tried to code into my website. Coding workarounds ended up adding unnecessary time to my digital project.
