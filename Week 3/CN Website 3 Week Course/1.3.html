Unit 1, Outcome 1

1.3 - Plan and Carry Out Tasks Using IT

I have created a Trello board which will be integral to planning and developing my digital project, which will be a website. The first step to creating my website will be to plan and create the content. The next step will be to develop the website using HTML code. After implementing HTML, the next step will be to style my website using CSS. The final step will be to test the functionality of my website, and make any last amendments.

Trello is a fantastic resource that will allow me to plan each step of my digital project by utilising lists. Within my Trello board, I will have three lists - a ‘to-do’ list, a ‘pending’ list and a ‘done’ list. Each list has cards which can be added and moved from list to list, and these cards detail each of the tasks of the digital project. Cards can be organised by using labels, and then colour coded. I have utilised labels in my planning process to help keep things organised. Another feature that I have utilised is the ‘due date’ function. Each task can be given a due date, and Trello will alert you if the due date for a task is due soon or overdue. Trello boards can be personalised to suit the needs of the project, and can be used alone or with a team of people. Comments can be added to each card, and in the comments section I have input the content for each of my questions. [13]
