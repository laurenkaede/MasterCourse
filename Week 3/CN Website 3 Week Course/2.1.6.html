2.1.6 - Describing Access Issues That Need Taking Into Account

Those with disabilities may face many issues when trying to access the internet and web pages. Those with visual impairments who use a screen reader may not be able to view images clearly. One way to combat this issue is to use alt tags. Alt tags can be assigned words to describe the image, and when a user hovers over an alt tag those words are read aloud. Video transcripts and subtitles are a great way to help users who have difficulty watching videos online, and will make websites more disability-friendly. Underlining links and creating a colour contrast between links and non-links will help those who are colour blind to easily find any links on a website. Ensuring that colours donâ€™t clash is another way to help those with disabilities read web pages better. Typically, many web pages will place black text onto a white background to ensure that all can read any content. Making sure that written content is formatted in an easy to read way, utilising grammar and correct spelling, is another way which can help those with disabilities access web pages more easily. An accessibility guide available on a web site can help a disabled user get the most out of any features which could make accessibility easier for them. [29] [30] [31] [32]
