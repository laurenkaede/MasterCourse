2.1.7 - Describe Which are the Appropriate Filetypes for Websites

When creating a website, it is important to use the appropriate file types so that the website can be developed and then function properly. HTML is a HyperText Markup Language that allows you to create websites. It consists of a series of short codes called tags, which can then be translated and read through a browser. CSS is known as a Cascading style sheet. This allows the coder to style their HTML code so that their website looks the way that they want it to. This may include changing font colours, creating margins and resizing images, etc. JPEG or PNG are the file formats that images utilise. In order to show an image on a website, the images need to be saved under one of these file types. MP3 file types allow websites to play sound files, whilst MP4 file types allow websites to stream videos. [33]
